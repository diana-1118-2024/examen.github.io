• ¿Qué es Cloud Computing?

    Cloud computing o cumputaciòn en la nube es un modelo de computaciòn en la nube, el cual brinda servicios a travès del internet/p>

  • ¿Qué es SaaS, PaaS e IaaS?

    Saas: Modelo de software el cual podemos encontrar servicios de aplicaciones permitiendo ejer¡cutar al usuario/p>

    PaaS: Proporciona sistemas operativos como desarrollo de programas el cual el proveedor se encarga de dar mantenimiento mediante la infraestrucutura/p>

    IaaS: Modelo de hardware que brinda servicios mediante la infraestructura/p>

    La virtualizaciòn es cuando podemos usar diferentes sistemas operativos dentro de un mismo servidor fìsico, tàmbien podemos ejecutar entorno de prueba sin afectar al servidor original/p>
